# 保险箱 - 移动密码管理器

一个专为移动设备设计的现代化密码管理器应用，具有简洁的界面和强大的功能。

## 功能特性

### 🛡️ 核心功能
- **安全存储**：本地加密存储密码，保护您的隐私
- **密码生成**：强大的随机密码生成器，可自定义长度和字符类型
- **分类管理**：支持个人、工作、社交媒体、财务等多种分类
- **收藏功能**：标记重要密码为收藏，快速访问
- **搜索筛选**：快速搜索和筛选密码条目

### 📱 移动端优化
- **响应式设计**：完美适配各种屏幕尺寸
- **触摸友好**：大按钮和直观的交互设计
- **离线使用**：无需网络连接，数据本地存储
- **PWA支持**：可安装为原生应用使用

### 🔒 安全特性
- **密码强度检查**：自动评估密码安全性
- **数据导出**：支持JSON格式导出备份
- **本地存储**：数据仅保存在用户设备
- **XSS防护**：内置HTML转义保护

## 快速开始

### 本地运行
1. 将项目文件复制到Web服务器目录
2. 通过浏览器访问 `index.html`
3. 或使用Python简单HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
   然后访问 http://localhost:8000

### 部署到服务器
1. 上传所有文件到Web服务器
2. 确保服务器支持HTTPS（PWA要求）
3. 访问应用URL即可使用

## 使用说明

### 添加密码
1. 点击右上角"+"按钮或"添加密码"快捷操作
2. 填写服务名称、用户名和密码
3. 选择分类和设置收藏状态
4. 点击"保存密码"

### 生成密码
1. 点击"生成密码"按钮
2. 调整密码长度和字符类型设置
3. 点击"使用此密码"或"复制密码"
4. 密码会自动保存到历史记录

### 管理密码
- **复制密码**：点击密码项右侧的复制按钮
- **编辑密码**：点击编辑按钮修改密码信息
- **删除密码**：点击删除按钮移除密码
- **筛选排序**：使用顶部筛选和排序功能

## 注意事项

1. **数据安全**：所有密码数据仅存储在浏览器本地，请定期备份
2. **浏览器兼容**：建议使用最新版本浏览器
3. **移动设备**：最佳体验在移动设备上
4. **图标文件**：需要添加实际的图标文件（icon-192.png, icon-512.png）
